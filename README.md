# Aurora Hub

Official Aurora website.

## How to Start

Double-click:

```text
Start_Aurora_Hub.bat
```

The launcher will check the environment, install dependencies if needed, start Aurora Hub and open the website automatically.

## How to Stop

Close the Vite server terminal.

Or run:

```text
Stop_Aurora_Hub.bat
```

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
```
